# SBDots Keybindings

| Keys | Action |
|------|--------|
| **Super + Q** | Close window |
| **Super + A** | Toggle floating |
| **Super + P** | Pseudo mode |
| **Super + F** | Fullscreen |
| **Super + S** | Split toggle |
| **Super + Shift + C** | Random wallpaper |
| **Super + Shift + P** | Color picker |
| **Super + Ctrl + R** | Reload Waybar |
| **Super + V** | Clipboard |
| **Super + Shift + S** | Screenshot |
| **Super + Shift + B** | Show keybinds |
| **Super + Shift + G** | Screen shader |
| **Super + Space** | App launcher |
| **Super + Shift + Q** | Power menu |
| **Super + E** | File manager |
| **Super + Return** | Terminal |
| **Super + Shift + E** | Emoji picker |
| **Super + Shift + Esc** | System monitor |
| **Super + Arrows** | Focus window |
| **Super + Tab** | Next workspace |
| **Super + Shift + Tab** | Previous workspace |
| **Super + [1–9]** | Switch workspace |
| **Super + Shift + [1–9]** | Move window to workspace |
| **Super + Ctrl + [1–9]** | Move all windows to workspace |
| **Super + Ctrl + Arrows** | Resize window |
| **Super + Mouse Left** | Move window |
| **Super + Mouse Right** | Resize window |
| **Brightness Keys** | Adjust brightness |
| **Volume Keys** | Adjust volume / mute |
| **Media Keys** | Next / Prev / Play / Pause |
